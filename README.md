# 985 力学考研招生数据站

基于 VitePress 的静态数据站，用于查看招收 `0801 力学`相关专业的 985 院校。除招生计划、复试人数和拟录取人数外，还提供逐院系的研究方向、初试科目、培养信息、复试细则、分数线、历年复录数据，以及 2026 年学校官网核验值和原文链接。

## 数据边界

- 当前字段来自“力学考研加油站”公开接口和详情页快照；接口不带年份，详情页把当前值列在 2026。
- 已逐一验证院校研究生招生或研究生院官网入口可访问。
- 官网入口只是核验入口；在没有定位到具体招生目录、复试名单或拟录取名单前，不把入口本身当作人数证据。
- 官网核验值与聚合参考值分开保存；名单合并多个专业或缺少专业代码时，只展示证据和说明，不强行计数。
- 缺失值显示为“待核验”，部分院系缺失时显示覆盖率，不按 0 人计算。
