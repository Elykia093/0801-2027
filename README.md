# 力学相关专业考研招生数据站

基于 VitePress 的静态数据站，用于查看 26 所 985 院校和中国科学院大学力学研究所的 `0801 力学`学硕，以及经官网明确核验的力学相关专硕。除招生计划、复试人数和拟录取人数外，还提供逐院系的研究方向、初试科目、培养信息、复试细则、分数线、历年复录数据，以及 2026 年学校官网核验值和原文链接。

## 数据边界

- 26 所 985 院校的39条0801记录来自“力学考研加油站”公开接口和详情页快照；国科大力学所4条0801记录来自国科大2026招生目录和力学所公示。
- 力学相关专硕不是全量0855：首批只纳入哈工大航天学院 `085500 机械` 中官网明确列出的11、42、61三个空天力学方向，共1条记录。
- 已逐一验证院校研究生招生或研究生院官网入口可访问。
- 官网入口只是核验入口；在没有定位到具体招生目录、复试名单或拟录取名单前，不把入口本身当作人数证据。
- 官网核验值与聚合参考值分开保存；名单合并多个专业或缺少专业代码时，只展示证据和说明，不强行计数。官网目录明确为0人的专业按已知0展示。
- 详情页统一展示 2023—2026 四年招生计划、分数与复录数据；来源缺失的年份保留待核验，不猜填。
- 来源页分数按拟录取初试平均/最高/最低分展示，不与官网复试线混用。
- 历史官网补证与聚合快照分开建模；当前已绑定北京航空航天大学、天津大学、国科大力学所、东北大学、湖南大学、浙江大学和武汉大学的2023—2025年度原文。
- 缺失值显示为“待核验”，部分院系缺失时显示覆盖率，不按 0 人计算。
- 44 条记录均有独立的复试参考书目收集状态；聚合页提示只作检索线索，只有打开官网原文或附件后才计入官网核验。
