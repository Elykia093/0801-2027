## 交付资料格式

每批资料尽量包含：

- 学校、学院、专业代码、年份。
- 官方原文标题、网页 URL 或原始 PDF/Excel/截图。
- 发布日期；不知道时可留空，但需保留原始文件或链接。
- 招生计划需说明公开招考、推免、专项、校区和共享计划口径。
- 名单需保留专业代码、初试总分、录取结果、批次及一志愿/调剂标识。
- 参考书目需提供复试科目、书名、作者/主编、版本、出版社、适用方向和官方出处；学校明确“不指定参考书”也应记录。

可直接按以下模板发送：

```text
学校：
学院：
专业代码：
年份：
资料类型：招生目录 / 复试细则 / 复试名单 / 拟录取名单 / 参考书目或考试大纲
原文标题：
官方链接或附件：
发布日期：
访问日期：
原文定位（页码 / 表格 / 附件名）：
能确认的数值与口径：
备注：
```

涉及参考书目时，另附以下区块；一本书一行，官网未提供的字段写“未标明”，不能自行补版本：

```text
参考书目状态：官网待核验 / 部分核验 / 已核验 / 官网明确不指定 / 官网未公开
复试科目代码：
复试科目名称：
适用研究方向：
书目1：书名｜作者/主编｜版本｜出版社｜出版年｜ISBN
书目2：
官网原文定位：
已检查但未采用的材料及原因：
```

工作流状态严格区分：

- `unreviewed`：尚未逐一打开官网材料；可以有聚合检索线索，但不能有官网来源或书目。
- `partial`：已打开官网原文或附件，但书名、作者、版本等仍不完整。
- `verified`：官网指定书目已完整结构化，并保留原文定位。
- `explicitly-not-designated`：官网原文明示不指定参考书。
- `not-published`：完成官网栏目、招生目录、复试细则和可见附件核查后仍未公开；必须保存已查原文，不能由空结果自动推断。

## 回填验收规则

- 官网首页、招生栏目首页只能作为检索入口，不能单独证明人数或书目。
- 合并多个专业且名单不含专业列时，只保存证据和说明，不拆分人数。
- 共享计划、专项计划、推免和统考需保留原始口径，不能直接相减猜值。
- 拟录取初试均分、最高分和最低分只能由可逐行核验的拟录取名单复算。
- 复试名单最低分不能冒充专业复试线。
- 参考书目只有跳转提示时标记“部分核验”；未读附件前不补书名、作者或版本。
- 学校明确不指定参考书时保存“明确不指定”，不标记为普通缺失。
- 每次回填后运行 `pnpm verify:data`；涉及字段或页面结构时再运行完整 `pnpm check`。
